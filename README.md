# Comment Service

## Requirements
- Java 8
- Maven
- Spring Boot
- Swagger 

## Install dependencies

To build the server 

```sh
$ mvn clean install
```

## Running Application

To start the server 

```sh
$ mvn clean spring-boot:run
```

## Api Rest
- Comments
    - Create a new comment

## Api Documentation

You can test API in http://localhost:8080/swagger-ui.html
